# Diabetes Risk Factor Analysis

## Overview
This project analyzes clinical diabetes data using Python, R, and SQL.
The goal is to identify important differences between diabetic and non-diabetic patients and demonstrate practical health informatics and data science skills.

## Technologies Used
- Python
- R
- SQL
- Pandas
- NumPy
- Matplotlib
- ggplot2
- SQLite

## Research Questions
1. Do diabetic patients have higher glucose levels?
2. Are diabetic patients older?
3. What modifiable risk factors may contribute to diabetes?

## Key Findings
- Diabetic patients have significantly higher average glucose levels.
- Diabetic patients are older on average.
- Several modifiable health indicators may be associated with diabetes risk.

## Dataset
The dataset contains patient clinical measurements including:
- Glucose
- Age
- Blood pressure
- BMI
- Diabetes outcome
